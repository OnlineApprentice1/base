# Section Order Variations

Don't use the same homepage section order as the last 3 builds. Check `registry.json` `sectionOrder` field before choosing.

## Available Patterns

### 1. Classic Flow
`hero → services → process → stats → testimonials → about-preview → cta`
Best for: straightforward service businesses where credibility builds linearly.

### 2. Story-First
`hero → about-preview → services → testimonials → process → cta`
Best for: businesses where the owner's story IS the differentiator (family-run, decades of experience).

### 3. Social-Proof-Heavy
`hero → testimonials → services → stats → process → cta`
Best for: businesses in competitive markets where trust is the #1 barrier.

### 4. Problem-Solution
`hero → pain-points → services → process → stats → testimonials → cta`
Best for: trades that solve urgent problems (plumbing, HVAC, roofing).

### 5. Portfolio-Led
`hero → gallery/portfolio → services → testimonials → about-preview → cta`
Best for: visual trades (landscaping, painting, metalwork, custom builds).

### 6. Minimal Impact
`hero → services → testimonials → cta`
Best for: simple businesses with few services. Let the work speak. 4 sections, not 7.

### 7. Authority Builder
`hero → stats → services → process → testimonials → faq-preview → cta`
Best for: businesses competing on expertise (electrical, engineering, specialized trades).

### 8. Community-Rooted
`hero → service-area-map → services → about-preview → testimonials → process → cta`
Best for: hyper-local businesses where geography is key (snow removal, lawn care).

### 9. Emergency-First
`hero-with-phone → services → availability/hours → process → testimonials → cta`
Best for: emergency trades (24/7 plumbing, locksmith, towing). Phone number prominent.

### 10. Premium Showcase
`hero → signature-project → services → process → materials/quality → testimonials → cta`
Best for: high-end trades (custom homes, luxury renovations, ornamental ironwork).

## Rules

1. **No repeat within 3 builds.** Check `registry.json` before choosing.
2. **Not every site needs 7 sections.** Patterns 6 and 9 are intentionally shorter. Vary length.
3. **Inner pages don't follow the homepage order.** Services, About, Contact each have their own structure.
4. **The hero is always first. The CTA is always last.** Everything between is variable.
5. **Record your choice** in the registry entry's `sectionOrder` field.
