# Design Concept Library

A pre-researched library of 40 design concepts for the website assembly line. Each concept is a complete visual direction — metaphor, colour, typography, hero treatment, and signature components — ready to be adapted to any Canadian trades business.

## Usage Instructions

### Before You Pick

1. **Read `registry.json`.** Parse the last 3 builds' archetypes, palettes, hero styles, signature moves, and fonts.
2. **Identify the current colour family rotation.** The differentiation engine groups palettes into four families: **Warm, Cool, Earth, Bold.** If the last build used Cool, pick from Warm, Earth, or Bold this time. The concepts below are already grouped by family.
3. **Don't repeat within 3 builds.** No concept name, hero technique, or signature component should match the last 3 registry entries.
4. **Brief drives the pick.** The vibe brief from Phase 0 narrows the field. A roofing company in Thunder Bay and a custom cabinet shop in Niagara need different visual languages. Let the brief's personality, trade, and location guide you.
5. **Adapt, don't copy verbatim.** Each concept is a starting point. Adjust the hue range, swap a font, or hybridise two concepts if the brief demands it. Document the adaptation in the registry.

### Quick-Pick Workflow

```
1. Check registry → identify forbidden family + recent concepts
2. Match brief personality → scan the right family category
3. Pick 2-3 candidates → compare hero techniques and signature components
4. Choose the one that best serves the trade + location + vibe
5. Record in registry with concept name in the archetype field
```

### 1. FORGE GLOW

**Metaphor:** The inside of a blacksmith's forge — molten metal cooling from white-hot to amber to dark iron. Heat radiating from the centre outward.

**Colour direction:** Warm / OKLCH hue 40-65. Deep charcoal base with amber-to-orange gradients. Primary: molten amber. Accent: white-hot highlight. Neutral: cooite ash grey.

**Font pairings:**
- Heading: **Syne** (geometric, industrial weight) + Body: **Outfit** (clean, modern)
- Heading: **Oswald** (condensed, muscular) + Body: **Source Sans 3** (neutral, readable)

**Hero technique:** Full-dark viewport with a single radial gradient expanding from centre, simulating forge heat. Business name in heavy condensed type, glowing at the edges with a subtle text-shadow pulse.

**Signature components:**
- **Heat-gradient progress bars** — service skill indicators that fill from deep red through amber to white, like heated metal
- **Anvil-stamp cards** — service cards with a heavy top border and a stamped/embossed effect on the icon, slight 1-2px inset shadow

**Best suited for:** Welding, metalwork, blacksmithing, fabrication, foundry work, heavy equipment repair.
