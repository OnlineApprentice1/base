# Placeholder Image Library

Curated Unsplash image URLs organized by trade category. Use these during builds instead of `placehold.co` solid-colour rectangles. These are still placeholders — the client replaces them with real photos — but they make the site look dramatically better during review.

**Usage pattern:**
```
https://images.unsplash.com/photo-{ID}?w={width}&h={height}&fit=crop&q=80
```

**QA note:** qa.sh warns about `placehold.co` but NOT about Unsplash URLs. Add a note to the client handoff that all images need replacement.
